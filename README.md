# Student Registration System

## About My Project 

This is the project which is based on student registration where, I have used HTML, CSS and JavaScript.
The main purpose of project is to add student details such as name, id, email and their contact details which is shown in table format.
It also allows deleting student records.

## Features Added 
- Adding the details such as name,email,contact and id
- Allows deleting the records
- Displaying the records of students in table

## Technologies Used In This Project
- Used HTML for giving structure to the project
- Used CSS for styling the form
- Used JavaScript for adding more functionality to the project

## How You Can Run The Project 
1. You need to download this project folder
2. Then you can open this HTML file in any of the web browser
3. Then you can start filling the student details
